## Statut et provenance

Le corpus a été produit par Boris avec ChatGPT les 15 et 16 juillet 2026, puis importé par Codex dans `zegame-docs` le 16 juillet 2026. Son contenu conceptuel est conservé ; seuls son index et son manifeste ont été complétés pour inclure les notes 11 et 12.

Le corpus constitue une **base de conception candidate**, pas une spécification intégralement validée. En cas de contradiction, la note de convergence et les décisions explicitement actées dans `docs/vision/` prévalent jusqu'à nouvel arbitrage de Boris.

## Règle d'usage pour les agents

- Citer le document source lorsqu'une proposition vient du corpus.
- Ne pas transformer une hypothèse pédagogique en calcul psychométrique sans validation dédiée.
- Vérifier la note de convergence avant de proposer un modèle de données ou un algorithme de recommandation.
- Signaler toute nouvelle contradiction plutôt que choisir silencieusement une version.
