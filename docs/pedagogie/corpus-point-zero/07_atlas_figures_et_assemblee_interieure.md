# Atlas des Figures de Conscience et Assemblée Intérieure

## 1. Fonction de l’Atlas

L’Atlas rassemble des figures historiques, mythologiques, fictionnelles, contemporaines, religieuses, artistiques et populaires caractérisées par les sept Puissances.

Une figure ne constitue pas un diagnostic de personnalité. Elle est une **incarnation narrative située** d’une configuration du Moteur.

## 2. Profiler une phase, pas une personne entière

Une même figure peut disposer de plusieurs fiches : avant l’appel, pendant la crise, dans l’Ombre, après l’alchimisation, au sommet de son œuvre ou lors de son retrait.

Pour les personnes réelles contemporaines, le portrait doit rester interprétatif, sourcé, daté et assorti d’un niveau de confiance.

## 3. Fiche d’une figure

- identité, univers, époque et phase ;
- type : réel, mythologique, fictionnel ;
- signature du Moteur ;
- capacités manifestées ;
- contextes où la circulation est libre ou bloquée ;
- Ombres, prix payé et conséquences ;
- chemin initiatique ;
- héritage et Transcendance ;
- sources et niveau de confiance.

## 4. Rejoindre une figure

Le joueur choisit une figure comme étoile polaire. L’outil compare les profils et calcule non une simple distance, mais un itinéraire ordonné dans le graphe du Moteur.

Exemple :

```text
Joueur : Imagination O1–L1 libre
Figure : Imagination O2–L3 libre
```

Chemin possible :

1. explorer L2 Projection ;
2. retrouver une circulation libre avec O1 ;
3. explorer O2 Conformité et passage à l’échelle ;
4. stabiliser O2–L2 ;
5. explorer L3 ;
6. transférer la capacité dans une œuvre.

## 5. Créer son propre rôle modèle

Le joueur peut composer une **Figure de Devenir** à partir de fragments de plusieurs figures : la Volonté de l’une, l’Imagination d’une autre, l’Émotion d’une troisième, etc.

L’avatar reçoit :

- un nom ;
- une image ;
- un récit d’origine ;
- une devise ;
- une signature du Moteur ;
- des pouvoirs ;
- une blessure ;
- une tentation ;
- un contrepoids ;
- un rite d’entrée et un rite de sortie.

## 6. Quatre rôles modèles

### Figure de Lumière non réalisée

Elle porte une Puissance que le joueur admire mais ne se reconnaît pas encore capable d’habiter.

### Figure d’Ombre réprimée

Elle incarne un pouvoir de limite, de service, de retrait ou de dissolution que le joueur confond avec sa forme captive.

### Figure d’Alliance

Elle montre comment deux polarités connues séparément peuvent devenir les organes d’un seul mouvement.

### Figure de Passage

Elle incarne le prochain déplacement accessible plutôt qu’un idéal ultime.

## 7. Assemblée Intérieure

Le joueur peut créer plusieurs avatars correspondant à des fonctions : Désirant, Souverain, Créateur, Cœur, Messager, Voyant, Donateur.

Chaque Puissance peut aussi disposer de plusieurs sièges : Lumière, Ombre, Source et forme blessée.

Questions de l’Assemblée :

- Qui parle actuellement en moi ?
- Qui n’est jamais consulté ?
- Quelle figure prend tout le pouvoir ?
- Quelle figure manque pour traverser la situation ?

## 8. Antagoniste intérieur

Le joueur peut créer une figure réunissant les capacités qu’il rejette : domination, ambition, séduction, froideur, conformité, doute, silence, sacrifice ou plaisir.

Le parcours vise à :

1. reconnaître sa fonction protectrice ;
2. extraire ses pouvoirs ;
3. distinguer la capacité de sa pathologie ;
4. l’intégrer dans l’Assemblée ;
5. lui donner une fonction et des limites.

## 9. Dépasser le rôle modèle

Le rôle modèle doit rester provisoire. Une étape avancée consiste à « tuer le maître » : distinguer ce qui a été reçu, ce qui a été imité, ce qui appartient désormais au joueur et ce qui doit être abandonné.

## 10. Génération de parcours par figure

```text
Profil actuel
→ configuration cible de la figure
→ calcul des écarts
→ chemin dans le graphe
→ expériences existantes
→ assemblage personnalisé
→ génération des chaînons manquants
→ expérimentation
→ mise à jour du profil
→ enrichissement du Commun
```

## 11. Garde-fous

- ne pas identifier une personne réelle à partir d’une photo ;
- ne pas présenter une caractérisation comme un diagnostic certain ;
- citer les sources publiques utilisées ;
- distinguer la personne historique, sa légende et la phase choisie ;
- inclure les Ombres, les coûts et les limites ;
- rappeler que le joueur ne cherche pas à devenir quelqu’un d’autre.
